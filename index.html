Sobre o projeto

O Link Snip é um projeto realizado por mim, Lucas Cordeiro. Trata-se de um encurtador de links de código aberto, foi utilziada a API da bitly para realizar o processo, sinta-se a vontade para usufruir da minha ferramenta.
